    1. What is Semantic HTML? 
    HTML that gives meaning to the content. Such as nav, section, footer, h1-h6, etc. 
    
    2. What is HTML used for? 
    HTML is a Markup Language that is used to give structure to a web page.
   
    3. What is an attribute and where do we put it? 
    An atribute is used to provide additional information about an HTML element. It goes inside the element tag and usualy has a name/value 
    pairing. IE: href is the attribute in <a href="">
  
    4. What is the h1 tag used for? How many times should I use it on a page?
    h1 defines the most inportant heading and should only be used once on a page
  
    5. Name two tags that have required attributes

    6. What do we put in the head of our HTML document? 
    Metadata like the <title> displayed in the window bar and <style> to define style (CSS) information for a single HTML page
   
    7. What is an id? 
    an id is an attribute for an HTML element that is unique to that specfic element and can be used by CSS to style that specfifc element 
    or JS to target that specific element
   
    8. What elements can I add an id to? 
    any HTML element
   
    9. How many times can I use the same id on a page? 
    once
   
    10. What is a class? 
    A class is an HTML attribute used to define styles with CSS for any elements that have that class name
   
    11. What elements can I add a class to? 
    Any HTML element
   
    12. How many times can I use the same class on a page? 
    as many as you want
   
    13. How do I get my link to open in a new tab?
    target="_blank"
   
    14. What is the alt attribute used for? 
    provides an alternate text for an image that is used to describe the image in cases where it doesnt load or isnt supported by what the 
    user is using to view the web page

    16. What is the difference between a section and a div?
    <section> is semantic HTML and also gives meaning to the content where <div> is used just for styling the content
